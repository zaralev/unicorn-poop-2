  • Wordpress Theme Design

    Final project from my Content Management Systems course. Created a child theme using PHP, HTML5, and CSS. Project began with wireframes, digital mockups, and then high fidelity screens which the final design was based off of. The goal of the assignment was to use PHP to generate posts and create pages rather than hard-coding.

  • Arctic Snack Web Game

    This game was created for my Web Game Design course with the intent of understanding HTML5 canvas and Phaser.io JavaScript framework, to create and interesting game with a timer, way to generate points, and win/lose screens. I created all custom art and sprites. This is the result of about a month of work, and with free time I hope to fully finish the game.

  • JavaScript Ping Pong

    Midterm assignment for Web Game Design. Created using vanilla JavaScript and HTML5 canvas, loosely following a tutorial during class, as well as extra credit features I had to code on my own (randomly colored sparks, custom sound, different paddle orientation).

  • Happy Sunshine App

    A team of three and I created a high fidelity prototype of a mobile application for the Happy Sunshine Food Truck, created on UXPin. Project involved guerrilla research, affinity map, business model canvas, mood boards, storyboarding, wireframes, paper prototype testing, and usability testing.
